# LAVENDAR SUPER CHATS
https://lavendarsuperchats.onrender.com/docs </br>
backend repo for a chat application using Perspective API for profanity check and postgres database to persist message.
## INTRODUCTION
The backend is run on uvicorn server with the help of gunicorn to manage instances of the app.
Deployment is on Render.com platform that acts as HAAS platform.
The postgres Database is elephantSQL.
The source code is wriiten fully in python 3.8

## GETTING STARTED
1. clone this repository
```
    git clone https://github.com/njeru-codes/chat-backend
```

2. create a virtual environment and install the dependencies
```
    python3 -m venv venv
```
3. create a env file in the root directiory of the project.
create a google account and enable the PERSPECTIVE API.
```
    DATABASE_URL="***************"
    SECRET_KEY='******************'
    APP_ID="*************"
    APP_KEY="**********"
    APP_SECRET="**********"
    GOOGLE_API_KEY="************"
```
4.  start the server
```
uvicorn app.main:app
```
